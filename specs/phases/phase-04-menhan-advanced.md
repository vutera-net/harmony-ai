# Phase 04: MenhAn - Advanced & Monetization

## Goals
- Tăng tỷ lệ giữ chân thông qua Destiny Journal.
- Tối ưu hóa doanh thu bằng gói Subscription.
- Tạo sản phẩm Viral thông qua PDF Report nghệ thuật.

## Prerequisites
- Master AI MVP đã hoạt động.
- Hệ thống thanh toán cơ bản đã xong.

## Task List

### TASK-04-00: Destiny Journal Implementation
**Mục tiêu**: Tạo vòng lặp tin cậy (Trust Loop) giữa AI và User.
**Depends on**: Phase 03
**Estimated scope**: L
**Việc cần làm**:
- [ ] Xây dựng UI Nhật ký vận mệnh (Timeline sự kiện).
- [ ] Implement logic "Đối soát": User nhập event $\rightarrow$ AI match với prediction.
- [ ] Xây dựng hệ thống Trust Score (Điểm tin cậy của AI).
- [ ] Viết prompt cho AI khi đối soát thành công (Tăng sự gắn kết).
**Definition of Done**:
- [ ] User có thể ghi nhật ký và nhận xác nhận "Đúng" từ AI.
